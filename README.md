# 🚗 E-Commerce Price Monitor & Trend Predictor

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/)
[![Data Source](https://img.shields.io/badge/Data--Source-WebScraper.io-orange.svg)](https://webscraper.io/test-sites/scroll)

An end-to-end Python web scraping and Machine Learning pipeline that monitors e-commerce vintage car listings, cleans raw unstructured details, performs exploratory analysis, and builds predictive models to forecast vehicle prices.

---

## 📌 Project Overview

This repository automates the pipeline of data collection, processing, and prediction:
1. **Scrapes** real-time vehicle details from the `webscraper.io` dynamic scrolling sandbox using dynamic page pagination detection.
2. **Cleans and structures** raw data into clean formats (extracting Title, Brand, Description, Year, Country, Mileage, Rarity Rating, Price, and Availability).
3. **Stores and exports** the processed data into local CSV storage (`data/scraped_cars.csv`) with optional support for PostgreSQL database export.
4. **Analyzes and visualizes** the listings, saving insights directly into the `plots/` directory.
5. **Predicts price trends** using a Scikit-Learn `LinearRegression` model.

---

## 🚀 Quick Start & Installation

### 1. Install Dependencies
Install all package requirements in your environment:
```bash
pip install -r requirements.txt
```

### 2. Run the Scraper
Harvest dynamic listings from the e-commerce sandbox:
```bash
python scraper.py
```

### 3. Execute Analysis & ML Training
Execute the analytical pipeline programmatically to run the EDA, generate all plots, and train the Machine Learning model:
```bash
jupyter nbconvert --to notebook --execute analysis.ipynb
```

## 📁 Repository Structure

*   `scraper.py` — Dynamic web scraper using requests and BeautifulSoup.
*   `analysis.ipynb` — Jupyter Notebook executing database exports, Seaborn visualizations, and regression modeling.
*   `data/` — Contains the output dataset `scraped_cars.csv`.
*   `plots/` — Diagnostic plots generated by the pipeline:
    *   `brand_share.png` — Market distribution per brand.
    *   `price_by_brand.png` — Price variation across vehicle brands.
    *   `price_by_year.png` — Vintage price progression over time.
    *   `price_vs_mileage.png` — Price impact of mileage.
    *   `actual_vs_predicted.png` — Regression model performance evaluation.
*   `requirements.txt` — Project dependencies configuration.
