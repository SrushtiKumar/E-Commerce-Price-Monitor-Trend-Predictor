# E-Commerce-Price-Monitor-Trend-Predictor
End-to-end Python ML pipeline scraping vehicle listings via BeautifulSoup. Structures messy data to generate a 90-day time-series dataset. Uses advanced feature engineering (lag/rolling windows) to train a Scikit-Learn Linear Regression model for next-day price forecasting, evaluated via MAE/RMSE and visualized with Seaborn.
